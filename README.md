# RPG Initiative Tracker Inserts

This app allows you to create printable inserts for initiative trackers for tabletop RPGs. You can add multiple inserts per A4 page, each with a front (player side) and back (DM side). All data is stored only in your browser.

## Features

- Add multiple inserts per page
- Player side: name and image (upload or URL)
- DM side: name, class, AC, max HP, passive perception, passive insight
- Printable A4 layout, each insert is 37x79mm
- No data is stored on a server

## Usage

1. Fill out the form to add inserts.
2. Upload or link an image for the player side.
3. Fill in DM-side details (class, AC, HP, passive perception, passive insight).
4. Click "Add Insert" to add the card to the page.
5. When ready, click "Print Inserts" to print the page (use A4 paper, 100% scale, no margins).

## Development

This project was built with [Vite](https://vitejs.dev/) and [React](https://react.dev/).

To run the development server:

```bash
npm run dev
```

To build for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
